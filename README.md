Banking Management System

gdb link: https://onlinegdb.com/C_9iItejW 

Description

A simple console-based Banking Management System developed in C++ using Object-Oriented Programming (OOP) concepts. The project supports multiple account types and basic banking operations.

Features
Create bank accounts
Deposit and withdraw money
Check account balance
Display account details
Calculate account interest
Overdraft support for checking accounts
Account Types
Savings Account
Checking Account
Fixed Deposit Account
OOP Concepts Used
Encapsulation
Inheritance
Polymorphism
Abstraction
Technologies
C++
Standard Library (iostream, string)
Compilation
g++ main.cpp -o banking
Run
./banking
Author

Developed as a C++ OOP practice project.
